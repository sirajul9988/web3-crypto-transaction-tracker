# Web3 Crypto Transaction Tracker

This is a simple Web3 project.  
It helps you track recent transactions of a wallet address.

This project is made for beginners who want to learn how blockchain transactions work.

## What This Project Does
- Read wallet address
- Fetch recent transactions
- Show transaction count
- Save transaction history to file

## Tools Used
- Node.js
- Simple JavaScript
- Public blockchain API (mock)
- Local JSON file for storage

## How To Install
1. Install Node.js
2. Download this repository
3. Open terminal in project folder
4. Run:
   npm install
5. Start the app:
   node app.js

## Folder Structure
- app.js = main file
- config.js = project settings
- tracker.js = transaction logic
- api.js = API calls
- saver.js = file saving
- data/ = saved transaction data

## Example Use
- Set wallet address
- App shows recent transactions
- App saves transaction list
